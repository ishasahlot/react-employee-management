# React Employee Management App

A React-based Employee Management application featuring authentication, employee listing with search and salary sorting, detailed employee view, and webcam photo capture functionality.

## Features

- Login authentication (localStorage-based)
- Employee list with pagination
- Search by name
- Sort by salary
- Detailed employee profile view
- Webcam photo capture and download
- Protected routes
- Responsive UI

## Tech Stack

- React
- React Router
- Vite
- JavaScript
- CSS
